# Figure index

The figures below are selected author-generated visual summaries from the experiments accompanying the manuscript. They contain no source code, auxiliary-data records, geographic coordinates, or full-resolution simulation imagery.

| File | Content | Role in the study |
| --- | --- | --- |
| `mechanism_compensation.png` | Error-level sensitivity and scan-angle-dependent residuals | Evaluates scan-angle and rotation-axis error compensation |
| `dynamic_error_residuals.png` | Residual response to unmodeled dynamic disturbances | Characterizes the applicability boundary of the low-order model |
| `gsd_aware_weighting.png` | Along-/cross-track GSD variation and weighted-estimation statistics | Connects GSD anisotropy with the benefit of weighted estimation |
| `gcp_coverage_observability.png` | GCP scan-angle coverage, accuracy, and conditioning | Evaluates parameter observability under different angular constraints |
| `closed_loop_validation.png` | Image measurement errors and independent check-point results | Provides image-level closed-loop validation |
| `relative_geometric_distortion.png` | Absolute displacement, translation-removed residuals, and pair-distance errors | Evaluates local relative geometric consistency after correction |

The natural-scene orthorectification panels used in the manuscript are intentionally not distributed on this project page while third-party basemap redistribution terms are being reviewed.
